# Crypto Dashboard Analysis

## Project Overview

This project is an interactive Cryptocurrency Analysis Dashboard developed using Microsoft Excel. The dashboard provides insights into cryptocurrency market performance through data visualization, pivot tables, pivot charts, slicers, and comparative analysis.

The project helps users analyze crypto market trends, compare coins, identify liquidity patterns, track historical price movements, and discover low-budget investment opportunities.

---

## Features

### 1. Main Dashboard

* Total Market Capital
* Total Price of Coins
* Total Circulating Supply
* Highest Price Coin
* Top Coin Price
* Interactive charts and slicers

### 2. Coin Comparison

* Compare two cryptocurrencies
* Price comparison
* Volume comparison
* Market Capital comparison
* Circulating Supply comparison

### 3. Liquidity Analysis

* Top cryptocurrencies by liquidity
* Interactive price category filtering
* Pie chart visualization

### 4. Price Analysis

* Current Price vs Previous Price comparison
* Top cryptocurrency price changes
* Interactive filtering using slicers

### 5. Historical Insights

* Previous 7-Day Price Analysis
* Previous 24-Hour Price Analysis
* 1-Hour Change Percentage Analysis
* Historical trend visualization

### 6. Low Budget Insights

* Identify low-budget cryptocurrencies
* Average downfall analysis
* Best coin selection within a price range
* Interactive price-group filtering
